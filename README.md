# python-data-cleaning-lab
Data cleaning and basic analysis using Python and Pandas, focusing on handling missing data, format issues, and data validation.
## What I did
- Loaded a CSV dataset using Pandas
- Identified and handled missing values
- Removed duplicate records
- Converted date columns into a consistent format
- Calculated basic statistics (mean and median)
- Visualized cleaned data using Matplotlib

## Skills used
- Python
- Pandas
- Matplotlib
- Data cleaning
- Problem-solving

## What I learned
- How to systematically clean and validate raw data
- The importance of consistent data formats
- How data issues can affect analysis and decision-making
